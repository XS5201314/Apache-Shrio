# 这是一个安全框架开发的企业管理系统
# 采用的技术

# 后端
IOC容器：Spring
Web框架：SpringMVC
ORM框架：Mybatis
安全框架：Shiro
数据源：durid
日志：log4j
前端框架：Bootstrap、ztree、assets

# 前端
缓存：redis
加密：MD5
数据库：MySQL8.15
工具：idea
Jdk：1.8
tomcat：8.0

# 其他
采用瀑布模型
插件：generator
测试工具：postman
数据库可视化：navicat
